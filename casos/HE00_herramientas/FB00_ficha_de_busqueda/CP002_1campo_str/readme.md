
## HE00FB00CP0000 Busqueda de un campo en una base de datos

### Descripción

Cargar la base de datos, **arena2_c.mv**, sin informcion geografica, y comprobar que se puede realizar una busqueda sobre una de sus columnas utilizando busqueda por atributo. todas las funciones probadas funcionan segun los resultados que se esperan


### Prerrequisitos

1. Tener instalado *gvSIG desktop 2.5.1.* 
2. Tener acceso a la base de datos **arena2_c.mv**
3. Tener arrancada la aplicacion con una vista nueva y vacia activa

### Pasos

1. Mostrar la tabla **ARENA_AC_VE_CO_PA** en la vista
2. Seleccionar menu Tabla/Busqueda por atributos. Al hacerlo aparecera una ficha con 200 elementos
3. Seleccionaremos el campo "AC_CALLE_NOMBRE"
4. Seleccionaremos el operador "Igual a"
5. Seleccionaremos el valor "JAIME BALMES"
6. Pulsaremos en el boton "Buscar"
